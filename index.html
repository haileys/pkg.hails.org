Debian

Using this repository

  • Import signing key
    • # curl https://pkg.hails.org/repo.pub | gpg --no-default-keyring --keyring /etc/apt/keyrings/pkg.hails.org.gpg --import
  • Add apt source, replacing distro with release codename, eg. bookworm
    • # echo 'deb [signed-by=/etc/apt/keyrings/pkg.hails.org.gpg] http://pkg.hails.org/debian/distro/ ./' > /etc/apt/sources.list.d/pkg.hails.org.list
